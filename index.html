adroit-age

This gem is used to find the age using date of birth, its also works for leap years.

Installation

gem install adroit-age

Insert into your gem file

gem "adroit-age"

Usage

require 'adroit-age'

dob =  Date.new(1990,1,23)
or
dob = "23/01/1990".to_date

age = dob.find_age
#=> 23
